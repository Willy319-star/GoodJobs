# Supabase integration

This folder contains the Supabase client helpers for Next.js App Router.

- `client.ts` is for browser/client components.
- `server.ts` is for server components, server actions, and route handlers.
- `env.ts` centralizes required environment variable checks.

Step 4 only adds the infrastructure. Step 5 will use these helpers for Supabase Auth, and Step 6 will use them for application CRUD.
